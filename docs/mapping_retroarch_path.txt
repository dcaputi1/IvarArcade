# COMPLETE BUTTON MAPPING - PHYSICAL TO RETROPAD TO MAME
# This documents the RetroArch (ra) path: Physical → Joystick → RetroPad → MAME

## RETROPAD STANDARD LAYOUT (Reference)
RetroPad buttons that map to MAME controls:
- a = RetroPad A button
- b = RetroPad B button  
- x = RetroPad X button
- y = RetroPad Y button
- l = RetroPad L shoulder
- r = RetroPad R shoulder
- l2 = RetroPad L2 trigger
- r2 = RetroPad R2 trigger
- l3 = RetroPad L3 stick button
- r3 = RetroPad R3 stick button
- start = RetroPad Start
- select = RetroPad Select
- up/down/left/right = D-pad directions

----------------|-------------|-----------------|----------------------

START           | Button 5    | start           | P1_START (JOYCODE_1_START) / START1
Up              | Axis -1     | up              | P1_JOYSTICK_UP
Down            | Axis +1     | down            | P1_JOYSTICK_DOWN
Left            | Axis -0     | left            | P1_JOYSTICK_LEFT
Right           | Axis +0     | right           | P1_JOYSTICK_RIGHT

NOTE: cfg_ra uses JOYCODE_1_BUTTON1/BUTTON2 for P1_BUTTON2/BUTTON1 (swapped)
      Button mapping: 1→2, 2→1, 4→3, 3→4 per cfg_ra

## RETROPAD TO MAME CONTROL MAPPING (cfg_ra summary)
================================================================================

This section shows how RetroPad buttons map to MAME's P1 controls in cfg_ra:

RetroPad → MAME P1 (for JOYCODE_1/2/4/5):
- a → P1_BUTTON2 (note: swapped with b)
- b → P1_BUTTON1 (note: swapped with a)  
- x → P1_BUTTON3 or P1_BUTTON4 (varies by device)
- y → P1_BUTTON4 or P1_BUTTON3 (varies by device)
- l → P1_BUTTON5 (shoulder buttons)
- r → P1_BUTTON6 (shoulder buttons)
- start → P1_START and START1
- select → P1_SELECT and COIN1
- up → P1_JOYSTICK_UP
- down → P1_JOYSTICK_DOWN
- left → P1_JOYSTICK_LEFT
- right → P1_JOYSTICK_RIGHT

Key Observations:
1. cfg_ra swaps button 1 and 2 (a↔b) compared to physical layout
2. Multiple devices can trigger same MAME control (OR logic in cfg)
3. Some controls like COIN1 accept input from all P1 controllers
4. HAT inputs (HAT1UP/DOWN/LEFT/RIGHT) used for directional controls
5. AXIS inputs used for analog controls (pedals, wheels)
